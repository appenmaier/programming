Welche Ausgabe erwartest Du nach Ablauf des abgebildeten Codings?

## Coding

```java
byte a = 116, b = 59;
byte c, d, e, f;

c = (byte) (a & b);
d = (byte) (a | b);
e = (byte) (a ^ b);
f = (byte) (~a);

System.out.println("c: " + c);
System.out.println("d: " + d);
System.out.println("e: " + e);
System.out.println("f: " + f);
```
